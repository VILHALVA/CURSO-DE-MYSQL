# INSTALANDO FERRAMENTAS
Para instalar o MySQL usando o WAMPP ou o XAMPP, você pode seguir os passos abaixo. Primeiro, escolha qual desses pacotes você deseja usar, pois ambos incluem o MySQL, o servidor web Apache e o PHP em um único pacote de instalação.

## Instalando o MySQL com o XAMPP:
1. **Baixe o XAMPP**:

   - Acesse o site oficial do XAMPP em [https://www.apachefriends.org/index.html](https://www.apachefriends.org/index.html).
   - Escolha a versão apropriada para o seu sistema operacional (Windows, macOS ou Linux) e faça o download do instalador.

2. **Execute o Instalador**:

   - Execute o arquivo de instalação que você baixou.
   - Durante o processo de instalação, você poderá selecionar os componentes que deseja instalar. Certifique-se de marcar a opção "MySQL" ou "MariaDB" (que é uma alternativa ao MySQL) para instalar o servidor de banco de dados.

3. **Configuração do MySQL**:

   - Após a instalação, você pode iniciar o servidor MySQL usando o painel de controle do XAMPP.

## Instalando o MySQL com o WAMPP:
1. **Baixe o WAMPP**:

   - Acesse o site oficial do WAMPP em [https://www.wampserver.com/en/](https://www.wampserver.com/en/).
   - Escolha a versão apropriada para o seu sistema operacional (Windows) e faça o download do instalador.

2. **Execute o Instalador**:

   - Execute o arquivo de instalação que você baixou.
   - Durante o processo de instalação, você poderá selecionar os componentes que deseja instalar. Certifique-se de marcar a opção "MySQL" para instalar o servidor de banco de dados.

3. **Configuração do MySQL**:

   - Após a instalação, você pode iniciar o servidor MySQL usando o painel de controle do WAMPP.

Ambos o XAMPP e o WAMPP são pacotes que incluem o MySQL, o Apache e o PHP, bem como outras ferramentas, para facilitar o desenvolvimento web local. Eles são úteis para criar e testar aplicativos web em seu computador antes de implantá-los em um ambiente de produção.

Lembre-se de que, após a instalação, você pode acessar o MySQL através do cliente MySQL, como o phpMyAdmin, que normalmente é incluído nesses pacotes. Certifique-se de configurar e gerenciar seus bancos de dados conforme necessário usando essas ferramentas.
